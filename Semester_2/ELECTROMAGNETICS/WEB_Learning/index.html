Μαθηματικά Εργαλεία

Διανύσματα σε Καρτεσιανό Σύστημα

Διάνυσμα: \(\vec{a} = a_x\hat{x} + a_y\hat{y} + a_z\hat{z}\)

Πράξεις Διανυσμάτων

Πρόσθεση: \(\vec{a} + \vec{b} = (a_x + b_x, a_y + b_y, a_z + b_z)\)

Εσωτερικό Γινόμενο: \(\vec{a} \cdot \vec{b} = a_x b_x + a_y b_y + a_z b_z = |\vec{a}||\vec{b}|\cos\theta\)

Εξωτερικό Γινόμενο: \(\vec{a} \times \vec{b} = |\vec{a}||\vec{b}|\sin\theta \hat{n}\)

Αριθμομηχανή Διανυσμάτων

Διαφορικοί Τελεστές

Απόκλιση (Divergence): \(\nabla \cdot \vec{F} = \frac{\partial F_x}{\partial x} + \frac{\partial F_y}{\partial y} + \frac{\partial F_z}{\partial z}\)

Στροβιλισμός (Curl): \(\nabla \times \vec{F} = \left(\frac{\partial F_z}{\partial y} - \frac{\partial F_y}{\partial z}\right)\hat{x} + \left(\frac{\partial F_x}{\partial z} - \frac{\partial F_z}{\partial x}\right)\hat{y} + \left(\frac{\partial F_y}{\partial x} - \frac{\partial F_x}{\partial y}\right)\hat{z}\)

Λαπλασιανός: \(\nabla^2 \phi = \frac{\partial^2 \phi}{\partial x^2} + \frac{\partial^2 \phi}{\partial y^2} + \frac{\partial^2 \phi}{\partial z^2}\)

όπου \(\varepsilon_0 = 8.854 \times 10^{-12}\) F/m

Πυκνωτές - Χωρητικότητα

Βασικοί Τύποι Πυκνωτών

1. Πυκνωτής Παράλληλων Πλακών

\[C = \frac{\varepsilon_0\varepsilon_r A}{d}\]

2. Κυλινδρικός Πυκνωτής

\[C = \frac{2\pi\varepsilon_0\varepsilon_r l}{\ln(b/a)}\]

3. Σφαιρικός Πυκνωτής

\[C = \frac{4\pi\varepsilon_0\varepsilon_r ab}{b-a}\]

Διαδραστικός Υπολογιστής

Οπτικοποίηση Ηλεκτρικού Πεδίου

Ενεργειακή Προσέγγιση

Ενέργεια αποθηκευμένη στον πυκνωτή:

\[U = \frac{1}{2}CV^2 = \frac{1}{2}QV = \frac{Q^2}{2C}\]

Πυκνότητα ενέργειας:

\[u = \frac{1}{2}\varepsilon_0\varepsilon_r E^2\]

Υπολογιστής Ενέργειας

Συγκριτική Ανάλυση Πυκνωτών

Τύπος Χωρητικότητα Πλεονεκτήματα Μειονεκτήματα
Παράλληλες Πλάκες \(C = \frac{\varepsilon A}{d}\) Απλή κατασκευή, ομοιόμορφο πεδίο Περιορισμένη χωρητικότητα
Κυλινδρικός \(C = \frac{2\pi\varepsilon l}{\ln(b/a)}\) Συμπαγής, μεγάλη χωρητικότητα Μη-ομοιόμορφο πεδίο
Σφαιρικός \(C = \frac{4\pi\varepsilon ab}{b-a}\) Συμμετρικό πεδίο Δύσκολη κατασκευή

Ρεύμα και Αντίσταση

Βασικός Νόμος του Ohm

Η θεμελιώδης σχέση:

\[V = I \cdot R\]

Πυκνότητα ρεύματος:

\[\vec{J} = \sigma \vec{E}\]

Διαδραστικός Υπολογιστής Ohm

Νόμοι του Kirchhoff

Νόμος Ρευμάτων (KCL)

\[\sum I_{εισόδου} = \sum I_{εξόδου}\]

Νόμος Τάσεων (KVL)

\[\sum V = 0\]

Προσομοιωτής Κυκλώματος

Ισχύς και Ενέργεια

Τρεις τρόποι υπολογισμού ισχύος:

\[P = V \cdot I\]

\[P = I^2 \cdot R\]

\[P = \frac{V^2}{R}\]

Υπολογιστής Ισχύος

I-V Χαρακτηριστικές Γραφήματα

Προχωρημένη Ανάλυση

Εξίσωση συνέχειας:

\[\nabla \cdot \vec{J} + \frac{\partial \rho}{\partial t} = 0\]

Νόμος Joule (πυκνότητα ισχύος):

\[p = \vec{J} \cdot \vec{E} = \sigma E^2\]

Μαγνητοστατική

Νόμος Biot-Savart

Μαγνητικό πεδίο από στοιχειώδες ρεύμα:

\[d\vec{B} = \frac{\mu_0}{4\pi} \frac{I d\vec{l} \times \vec{r}}{r^3}\]

Ολοκληρωτική μορφή:

\[\vec{B} = \frac{\mu_0 I}{4\pi} \oint \frac{d\vec{l} \times \vec{r}}{r^3}\]

3D Οπτικοποίηση Μαγνητικού Πεδίου

Νόμος του Ampère

Ολοκληρωτική μορφή:

\[\oint \vec{B} \cdot d\vec{l} = \mu_0 I_{enclosed}\]

Διαφορική μορφή:

\[\nabla \times \vec{B} = \mu_0 \vec{J}\]

Υπολογιστής Πεδίου Ampère

Μαγνητικές Δυνάμεις

Δύναμη σε κινούμενο φορτίο:

\[\vec{F} = q(\vec{v} \times \vec{B})\]

Δύναμη σε αγωγό:

\[\vec{F} = I\vec{L} \times \vec{B}\]

Δύναμη μεταξύ αγωγών:

\[\frac{F}{L} = \frac{\mu_0 I_1 I_2}{2\pi d}\]

Υπολογιστής Μαγνητικής Δύναμης

Μαγνητική Ροή

Ορισμός:

\[\Phi_B = \int \vec{B} \cdot d\vec{A}\]

Νόμος Gauss για μαγνητισμό:

\[\oint \vec{B} \cdot d\vec{A} = 0\]

Δεν υπάρχουν μαγνητικοί μονόπολοι!

Οπτικοποίηση Μαγνητικής Ροής

Εξισώσεις του Maxwell

1. Νόμος Gauss (Ηλεκτρισμός)

Ολοκληρωτική:

\[\oint \vec{E} \cdot d\vec{A} = \frac{Q_{enc}}{\varepsilon_0}\]

Διαφορική:

\[\nabla \cdot \vec{E} = \frac{\rho}{\varepsilon_0}\]

2. Νόμος Gauss (Μαγνητισμός)

Ολοκληρωτική:

\[\oint \vec{B} \cdot d\vec{A} = 0\]

Διαφορική:

\[\nabla \cdot \vec{B} = 0\]

3. Νόμος Faraday

Ολοκληρωτική:

\[\oint \vec{E} \cdot d\vec{l} = -\frac{d\Phi_B}{dt}\]

Διαφορική:

\[\nabla \times \vec{E} = -\frac{\partial \vec{B}}{\partial t}\]

4. Νόμος Ampère-Maxwell

Ολοκληρωτική:

\[\oint \vec{B} \cdot d\vec{l} = \mu_0(I + I_d)\]

Διαφορική:

\[\nabla \times \vec{B} = \mu_0\vec{J} + \mu_0\varepsilon_0\frac{\partial \vec{E}}{\partial t}\]

Διαδραστική Επίδειξη Σύζευξης Πεδίων

Παραγωγή Εξίσωσης Κύματος

Από τις εξισώσεις Maxwell προκύπτει:

\[\nabla^2 \vec{E} - \mu_0\varepsilon_0 \frac{\partial^2 \vec{E}}{\partial t^2} = 0\]

\[\nabla^2 \vec{B} - \mu_0\varepsilon_0 \frac{\partial^2 \vec{B}}{\partial t^2} = 0\]

Ταχύτητα φωτός: \(c = \frac{1}{\sqrt{\mu_0\varepsilon_0}} = 3 \times 10^8\) m/s

Ηλεκτρομαγνητικά Κύματα

Γραμμική Πόλωση

Ηλεκτρικό πεδίο ταλαντώνεται σε ένα επίπεδο:

\[\vec{E} = E_0 \cos(kz - \omega t) \hat{x}\]

\[\vec{B} = \frac{E_0}{c} \cos(kz - \omega t) \hat{y}\]

Γραμμικά Πολωμένο Κύμα

Κυκλική Πόλωση

Το ηλεκτρικό πεδίο περιστρέφεται:

\[\vec{E} = E_0[\cos(kz - \omega t) \hat{x} \pm \sin(kz - \omega t) \hat{y}]\]

Δεξιόστροφη (+) ή Αριστερόστροφη (-)

Κυκλικά Πολωμένο Κύμα

Ελλειπτική Πόλωση

Γενική περίπτωση πόλωσης:

\[\vec{E} = E_{0x}\cos(kz - \omega t) \hat{x} + E_{0y}\cos(kz - \omega t + \phi) \hat{y}\]

Διαφορά φάσης φ καθορίζει το σχήμα

Υπολογιστής Ιδιοτήτων Κύματος

Συμβολή Κυμάτων

Δύο κύματα που επικαλύπτονται:

\[E_{total} = E_1 + E_2\]

Εποικοδομητική: φάσεις συμφωνούν

Καταστρεπτική: φάσεις αντίθετες

Προσομοιωτής Συμβολής

Φαινόμενο Doppler

Μεταβολή συχνότητας λόγω κίνησης:

\[f' = f \frac{c \pm v_r}{c \pm v_s}\]

vᵣ: ταχύτητα δέκτη, vₛ: ταχύτητα πηγής

Ενέργεια και Φωτόνια

Διάνυσμα Poynting

Ροή ηλεκτρομαγνητικής ενέργειας:

\[\vec{S} = \frac{1}{\mu_0}(\vec{E} \times \vec{B})\]

Ένταση (μέση ισχύς ανά επιφάνεια):

\[I = \langle S \rangle = \frac{1}{2}\varepsilon_0 c E_0^2\]

3D Οπτικοποίηση Διανύσματος Poynting

Φωτοηλεκτρικό Φαινόμενο

Εξίσωση Einstein:

\[hf = \phi + E_{k,max}\]

φ: έργο εξόδου, Ek,max: μέγιστη κινητική ενέργεια

Κατώφλι συχνότητας:

\[f_0 = \frac{\phi}{h}\]

Προσομοιωτής Φωτοηλεκτρικού Φαινομένου

Πυκνότητα Ενέργειας

Ηλεκτρική:

\[u_E = \frac{1}{2}\varepsilon_0 E^2\]

Μαγνητική:

\[u_B = \frac{1}{2\mu_0} B^2\]

Συνολική:

\[u = u_E + u_B = \varepsilon_0 E^2\]

Υπολογιστής Πυκνότητας Ενέργειας

Ορμή Φωτονίων

Ορμή φωτονίου:

\[p = \frac{h}{\lambda} = \frac{E}{c}\]

Πίεση ακτινοβολίας:

\[P_{rad} = \frac{I}{c}\]

Για τέλεια απορρόφηση

Επίδειξη Ορμής Φωτονίων

Κυματο-Σωματιδιακή Διττότητα

Κυματικές ιδιότητες: Συμβολή, Περίθλαση, Πόλωση

Σωματιδιακές ιδιότητες: Φωτοηλεκτρικό, Compton, Ορμή

Σχέση de Broglie:

\[\lambda = \frac{h}{p}\]
