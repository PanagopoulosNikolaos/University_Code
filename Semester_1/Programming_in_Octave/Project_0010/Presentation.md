### 3. Παραγοντοποίηση LU (με πιθανή χρήση μεθόδου Gauss)

Η εντολή:

```matlab
[L, U, P] = lu(A);

```

δημιουργεί τους πίνακες L, U και P, όπου:

- L είναι κάτω τριγωνικός (lower triangular) με 1 στην κύρια διαγώνιο.
- U είναι άνω τριγωνικός (upper triangular).
- P είναι πίνακας μεταθέσεων (permutation matrix), ο οποίος εφαρμόζεται για την αντιμετώπιση αριθμητικών δυσχερειών (π.χ. όταν απαιτείται να αλλάξουν μεταξύ τους κάποιες γραμμές κατά την απαλοιφή).

Η σχέση που ισχύει είναι:

P * A = L * U

### 4. Επίλυση κάθε συστήματος με εμπρόσθια και οπίσθια αντικατάσταση

Αφού έχουμε τους τριγωνικούς πίνακες L και U, μπορούμε να λύσουμε το σύστημα P*A x = Pb (μεταθέτουμε πρώτα το διάνυσμα b για λόγους συνέπειας) σε δύο φάσεις:

1. Λύνουμε το L y = P*b (εμπρόσθια αντικατάσταση).
2. Έπειτα λύνουμε το U x = y (οπίσθια αντικατάσταση).

Στον κώδικα, αυτό υλοποιείται ως εξής:

### Για το b1

```matlab
y1 = SolForward(L, P*b1);
x1 = SolBackward(U, y1);

```

### Για το b2

```matlab
y2 = SolForward(L, P*b2);
x2 = SolBackward(U, y2);

```

Ουσιαστικά, λόγω της σχέσης P*A = L*U, η επίλυση A x = b μετατρέπεται σε P*A x = P*b ⇒ L*U x = P*b. Γι’ αυτό πρώτα λύνουμε L y = P*b και κατόπιν U x = y.

## Συνάρτηση SolForward(L, b)

Η συνάρτηση SolForward πραγματοποιεί την εμπρόσθια αντικατάσταση (forward substitution) για να λύσει το σύστημα L y = b όταν ο L είναι κάτω τριγωνικός. Ο αλγόριθμος λειτουργεί γραμμικά από την πρώτη έως την τελευταία μεταβλητή:

1. Αρχικοποιούμε το διάνυσμα λύσης y με μηδενικά.
2. Για κάθε γραμμή i:
    - Θέτουμε y(i) = b(i).
    - Αφαιρούμε τα γινόμενα L(i,j)*y(j) για j από 1 έως i-1, ώστε να αφαιρεθεί η ήδη γνωστή συμβολή των προηγουμένων μεταβλητών.
    - Διαιρούμε τελικά το αποτέλεσμα με το διαγώνιο στοιχείο L(i,i).

Ο κώδικας φαίνεται παρακάτω:

```matlab
function y = SolForward(L, b)
    n = length(b);        % Διάσταση του συστήματος
    y = zeros(n,1);       % Αρχικοποίηση διανύσματος y

    for i = 1:n
        y(i) = b(i);
        for j = 1:i-1
            y(i) = y(i) - L(i,j)*y(j);
        end
        y(i) = y(i) / L(i,i);
    end
end

```

### Ροή υπολογισμού

- Στον πρώτο βρόχο (i=1), ουσιαστικά y(1) = b(1)/L(1,1).
- Στον επόμενο βρόχο (i=2), y(2) προσαρμόζεται αφαιρώντας το γινόμενο L(2,1)*y(1) και διαιρώντας με L(2,2).
- Με παρόμοιο τρόπο συνεχίζει μέχρι την τελευταία συνιστώσα y(n).

## Συνάρτηση SolBackward(U, y)

Αντίστοιχα, η συνάρτηση SolBackward λύνει το U x = y, όπου ο U είναι άνω τριγωνικός. Εφαρμόζεται ο αλγόριθμος οπίσθιας αντικατάστασης (backward substitution):

1. Ξεκινάμε από την τελευταία εξίσωση (i = n) και προχωράμε προς τα πίσω.
2. Για κάθε γραμμή i (από κάτω προς τα πάνω):
    - Θέτουμε x(i) = y(i).
    - Αφαιρούμε τα γινόμενα U(i,j)*x(j) για j από i+1 έως n.
    - Διαιρούμε με το διαγώνιο στοιχείο U(i,i).

Ο κώδικας:

```matlab
function x = SolBackward(U, y)
    n = length(y);         % Διάσταση του συστήματος
    x = zeros(n,1);        % Αρχικοποίηση διανύσματος x

    for i = n:-1:1
        x(i) = y(i);
        for j = i+1:n
            x(i) = x(i) - U(i,j)*x(j);
        end
        x(i) = x(i) / U(i,i);
    end
end

```

### Ροή υπολογισμού

- Ξεκινάμε από i = n:
x(n) = y(n)/U(n,n).
- Μετά πηγαίνουμε στο i = n-1:
x(n-1) = ( y(n-1) - U(n-1,n)*x(n) ) / U(n-1,n-1).
- Συνεχίζουμε μέχρι το i=1.

## Η Συνάρτηση `toReducedEchelon(A, elementary_matrices)`

Η συνάρτηση `toReducedEchelon(A, elementary_matrices)` μετατρέπει τον κλιμακωτό πίνακα `A` σε ανοιγμένη κλιμακωτή μορφή.

    ```matlab

Για κάθε στοιχείο πάνω από το pivot, υπολογίζεται ένας `factor` και γίνεται η κατάλληλη γραμμοπράξη για να μηδενιστεί το στοιχείο.     ```matlab
    for i = 1:n
        factor = 1 / A(i, i);
        A(i, :) = A(i, :) * factor;
        E = eye(n);
        E(i,i) = factor;
        elementary_matrices{end+1} = E;
    end
    
    ```
    
    Για κάθε γραμμή, το pivot (το πρώτο μη μηδενικό στοιχείο) διαιρείται με τον εαυτό του για να γίνει ίσο με 1. Αυτό επιτυγχάνεται πολλαπλασιάζοντας ολόκληρη τη γραμμή με τον αντίστροφο του pivot. Η αντίστοιχη στοιχειώδης μήτρα είναι μια διαγώνια μήτρα με όλα τα διαγώνια στοιχεία ίσα με 1 εκτός από το στοιχείο στη θέση `(i, i)` που είναι ίσο με `factor`.
    

## Η Συνάρτηση `computeInverse(A, elementary_matrices)`

Η συνάρτηση `computeInverse(A, elementary_matrices)` υπολογίζει τον αντίστροφο του πίνακα `A` χρησιμοποιώντας τις στοιχειώδεις μήτρες.

1. **Αρχικοποίηση του Αντίστροφου:**
    
    ```matlab
    n = size(A, 1);
    inverse_A = eye(n);
    
    ```
    
    Ο αντίστροφος αρχικοποιείται ως η ταυτοτική μήτρα.
    
2. **Πολλαπλασιασμός των Στοιχειωδών Μητρών (σε Αντίστροφη Σειρά):**
    
    ```matlab
    for i = length(elementary_matrices):-1:1
        inverse_A = inverse_A * elementary_matrices{i};
    end
    
    ```
    
    Οι στοιχειώδεις μήτρες που αποθηκεύτηκαν κατά τη μετατροπή του `A` σε ανοιγμένη κλιμακωτή μορφή, αντιπροσωπεύουν τις στοιχειώδεις γραμμοπράξεις. Για να βρούμε τον αντίστροφο, πολλαπλασιάζουμε τις στοιχειώδεις μήτρες με αντίστροφη σειρά. Αυτό συμβαίνει επειδή αν $E_1 A = B$, $E_2 B = C$, τότε $E_2 E_1 A = C$. Για να λύσουμε για $A$, πολλαπλασιάζουμε με τους αντίστροφους των $E_i$ με αντίστροφη σειρά: $A = E_1^{-1} E_2^{-1} C$. Στην περίπτωσή μας, η ακολουθία των $E_i$ μετατρέπει τον $A$ στην ταυτοτική μήτρα $I$, οπότε $E_k ... E_2 E_1 A = I$, και $A^{-1} = E_k ... E_2 E_1$.

## Συνάρτηση `calculate_eigenvalues()`

```matlab
function [eigenvals, char_poly] = calculate_eigenvalues(A)
    % Έλεγχος αν ο πίνακας είναι 3x3 και συμμετρικός
    [m, n] = size(A);
    if m ~= 3 || n ~= 3
        error('Ο πίνακας πρέπει να είναι 3x3');
    end
    if ~isequal(A, A')
        error('Ο πίνακας πρέπει να είναι συμμετρικός');
    end

    % Υπολογισμός συντελεστών χαρακτηριστικού πολυωνύμου
    a = -trace(A);
    b = det(A(1:2,1:2)) + det(A(2:3,2:3)) + det([A(1,1) A(1,3); A(3,1) A(3,3)]);
    c = -det(A);
    char_poly = [1 a b c];

    % Υπολογισμός ιδιοτιμών χρησιμοποιώντας τη συνάρτηση roots
    eigenvals = roots(char_poly);
end

```

- `% Έλεγχος αν ο πίνακας είναι 3x3 και συμμετρικός`: Σχόλιο που περιγράφει τον σκοπό του επόμενου μπλοκ κώδικα.

- `if m ~= 3 || n ~= 3`: Ελέγχει αν ο αριθμός των γραμμών (`m`) ή ο αριθμός των στηλών (`n`) δεν είναι ίσος με 3. Ο `||` συμβολίζει την λογική πράξη "OR".
